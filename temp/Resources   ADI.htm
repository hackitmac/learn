ADI Resources Page

There are tons of different technologies out there, each of which has its own tutorials, documentation, and community (good or bad). ADI has done the hard work of tracking down the best resources, summarizing each technology, and identify critical concepts based on goal so that the only thing standing between you and the implementation of your idea is a little bit of learning and hard work.

How To Use This Website

Technology is really intimidating at first, but don't fear! We're here to guide you through. Figure out what your goal is (see the "paths" section) and look at all of the technologies listed under the path. Read the descriptions and check out the resources that sound most appealing. If you're having trouble or want to show something off, come to Cookies and Code on Wednesdays 10pm-12am in Lerner 569. You can also email us if you want help with anything on this page, it doesn't include your personal goals, or you have any suggestions.

That said, we can't teach you everything. You have to go out on your own to learn a lot of these things. This site is intended as a springboard for you to take the first steps; hopefully, from there everything else is easy.

General Tips

Do

Learn "Google-fu." This is the term we use for a problem-solving using Google. It involves being able to pick out the right keywords to search for and decide which resources are good and bad. Once you have this mastered, you'll be mostly independent. Never be afraid to ask for help, though! Most people who know more than you are happy to share.

Stack Overflow is a question and answer site that tends to have really good information. Plus, if you're having a problem that's not solved on there, you can ask a question yourself.

Don't

W3Schools tends to be one of the first results in Google searches, but often has out-of-date or misleading information. We recommend using the Mozilla Developer Network instead.

Technologies

HTML

Dictates the structure and contents of every web page on the internet. Lets you put text and images on a web page; if you want to make a web site, start here.
  • HTML Dog — Walks you through learning all the different HTML tags one at a time. Beginner, intermediate, and advanced tracks. A great place to start!
  • Mozilla Developer Network — Extensive reference by the organization behind Firefox. If you need any element of HTML explained to you very clearly or in detail, look here.
  • Dive Into HTML5 — Clearly explains hand-picked features of the latest and greatest version of HTML, HTML5, and how HTML5 compares to earlier HTML standards.
  • HTML5 Rocks — Promotes innovations and describes best practices in HTML5. Once you've mastered the basics, check in here regularly.
  • HTML5 Boilerplate — The web's most popular front-end template. Starting a new project? Hit the ground running with HTML5 Boilerplate.
  • Codecademy Web Fundamentals Track — Interactive, online tutorial to learn to HTML and CSS. No experience necessary, and great for learning HTML and CSS together.

    • CSS

    The code that styles HTML: dictates look, layout, and general presentation. Once you've laid out your content using HTML, use CSS to make it look good.
  • HTML Dog — HTML dog will walk you through learning a lot of the most important CSS properties. They have beginner, intermediate, and advanced tracks.
  • Mozilla Developer Network — Extensive reference by the organization behind Firefox. If you need any property of CSS explained to you very clearly or in detail, look here.
  • Codecademy Web Fundamentals Track — Interactive, online tutorial to learn to HTML and CSS. No experience necessary, and great for learning HTML and CSS together.

    • CSS Preprocessors

    Once you've got the hang of CSS, CSS preprocessors allow you to write CSS using more powerful features, such as variables, functions, and “mixins.” After development, these files get compiled into regular CSS that all web browsers can understand. Makes CSS easier.
  • Sass — Sass makes CSS fun again. Sass is an extension of CSS3, adding nested rules, variables, mixins, selector inheritance, and more. It’s translated to well-formatted, standard CSS using the command line tool or a web-framework plugin.
  • LESS — The dynamic stylesheet language. LESS extends CSS with dynamic behavior such as variables, mixins, operations and functions.
  • Stylus — Stylus is a revolutionary new language, providing an efficient, dynamic, and expressive way to generate CSS. Supporting both an indented syntax and regular CSS style.
  • CSS Preprocessor Shootout — Side-by-side comparison of Sass, LESS, and Stylus. Check here if you want to see the differences.

    • JavaScript

    The programming language of the web. If you want your web site/app to move, change, or communicate, learn JavaScript.
  • Eloquent JavaScript — Eloquent JavaScript is designed for people without any programming experience, and will teach you how to program while also teaching you the ins and outs of JavaScript.
  • Codecademy JavaScript Track — Interactive, online tutorial to learn to JavaScript. Best taken after Codecademy's Web Fundamentals track.
  • codeschool JavaScript Road Trip — An introduction to the very basics of the JavaScript language. Build a foundation of JavaScript syntax and learn how to use values, variables, and files.
  • LearnStreet beginner course on JavaScript — In this course, you'll learn how to code in JS, a flexible and powerful programming language used all over the web.
  • Airbnb JavaScript Style Guide — This page is Airbnb's JavaScript style guide, and is a great reference for writing quality JavaScript code. They include suggestions for spacing, variable declarations, and pretty much everything else JavaScript.

    • jQuery

    A popular (used on more than half of the 10,000 most popular websites) JavaScript library that greatly simplifies client-side scripting. Write less, do more. If you use JavaScript, jQuery will give you cool features and make your life easier.
  • jQuery — The documentation on the jQuery official site is superb, and is a great way of getting started learning.
  • 30 Days to Learn jQuery — A video series which emails you a video once a day for 30 days in order to provide step-by-step instruction in learning to use jQuery.
  • Unheap — Unheap is an aggregation of open-source jQuery plugins (meaning they’re free to use!), which allow you to do some amazing things like animated image galleries and scroll to refresh.
  • Codecademy jQuery Track — Interactive, online tutorial to learn to jQuery. Best after taking the Javascript track.
  • Try jQuery — A combination of videos, interactive console challenges, and writing code in the browser. You'll need about 3 hours of time to make it through everything.

    • Bootstrap

    Twitter's front-end UI framework which packages up a bunch of useful CSS and JavaScript tools. Great for making web sites that look pretty good (if a little generic) quickly. It's "responsive", so your site will look good on screens of all sizes (desktop, tablet, mobile).
  • Bootstrap — Download and learn Bootstrap. See especially the "Getting started" page, and check out some examples.
  • Bootstrap Magic — Easily create a Bootstrap theme.
  • 1pxdeep — Flat-ish version of Bootstrap.
  • Flatstrap — Another flat version of Bootstrap. Looks similar, but with gradients and shadows gone. Also check out the "flatstrap-sass" and "flatstrap-rails" Ruby gems (easy to find on Google) for painless integraiton with Ruby on Rails (if you don't know what Rails is, disregard that last bit).

    • Foundation

    A front-end UI framework similar to Bootstrap. Looks a little different. Great for getting a site up and running quickly. It's "responsive", so your site will look good on screens of all sizes (desktop, tablet, mobile).
  • Foundation — Download and learn Foundation. See especially the "Docs" page.

    • AngularJS

    A front-end framework for building maintainable one-page applications. Lets you "extend" HTML for dynamic applications. If you have enough JavaScript going on that things are starting to get complicated, look into Angular.
  • Introduction to AngularJS — Post about the capabilities and usage of angular. Good overview of why you might want to use it.
  • AngularJS Guide — Official documentation from the AngularJS team. A great starting point.
  • Angular Seed — A small skeleton AngularJS app.
  • AngularJS Learning — A bunch of links to blog posts, articles, videos, etc. for learning AngularJS.
  • ADI Schedule Builder — ADIs tool for planning your semester's courses. Built on AngularJS.

    • Backbone.js

    An MVC framework for frontend JavaScript. Backbone.js gives structure to web applications. If you have enough JavaScript going on that things are starting to get complicated, look into Backbone.
  • Backbone.js — Backbone home page.

    • CoffeeScript

    A frontend language meant to solve the various problems/warts in JavaScript. Syntax inspired by Ruby. If you don't like JavaScript syntax but want to code for the browser, check out CoffeeScript.
  • CoffeeScript — CoffeeScript homepage. Side-by-side comparison with JavaScript.

    • Python

    A general-purpose, high-level programming language focused on clarity and readability. Often used to write the backend for web applications. A great first programming language.
  • Learn Python the Hard Way — Learn Python the Hard Way is a book (available for free here) for people who do not have experience programming, and it will teach you to program while teaching you Python.
  • Codecademy Python Track — Interactive, online tutorial to learn to program Python. No experience necessary.
  • Dive into Python — Dive into Python is a book about Python for people who already know how to program in another language.
  • LearnStreet beginner course on Python — Online intro to Python 2.7. Will teach you the key concepts in Python. No experience necessary.

    • Flask

    A simple and lightweight web framework for Python that allows you to quickly develop web applications. This is the easiest way to get going with a simple web app. If you want to extend your web site to have changing, dynamic content, Flask is a great place to start.
  • Flask — The Flask homepage. Gives a quick introduction, and the docs are really good.

    • Django

    A one-stop MVC web framework for Python that brings together all the bits you need to develop a web application. It's a lot heavier than Flask, but might be better for more complicated apps.
  • Django — The Django homepage. Has more information

    • Ruby

    A general-purpose, high-level programming language focused on clarity and readability. Often used to write the backend for web applications. Similar to Python, but most people prefer one to the other (ADI right now uses more Python).
  • Codecademy Ruby Track — Interactive, online tutorial to learn to program Ruby. No experience necessary.
  • Why's (Poignant) Guide to Ruby — Part tutorial, part comic book, part really bizarre art, Why's (Poignant) Guide to Ruby is considered a classic way to learn the language.
  • Code School Ruby — Learn the basic building blocks of Ruby, all in the browser.
  • Ruby User's Guide — Another, more traditional way to learn Ruby. Try this if Why's isn't your style.
  • Official Ruby Documentation — Official source of information about the Ruby language.
  • Mr. Neighborly's Humble Little Ruby Book — From the website: "An up to date book on Ruby programming, written in a style described as 'a beautiful display of pragmatically chunky bacon, wrapped in a nutshell.' Or something like that."
  • LearnStreet Ruby for Beginners V3 — In this course, you'll learn how to code in Ruby, a flexible yet powerful programming language that’s popular over the web.

    • Ruby on Rails

    Ruby on Rails is a model-view-controller framework for building web applications with Ruby. As such, it touches all parts of the stack, from running a web server to rendering front-end templates. A really popular framework for web apps.
  • Official Ruby on Rails Tutorial — This tutorial will walk you through building your first app with Ruby on Rails.
  • Rails Tutorials, Examples, and Starter Applications — A compilation of starter applications, tutorials, templates, and examples for different Rails libraries.
  • Code School Rails for Zombies — Learn Ruby on Rails, an open-source framework for the Ruby programming language. Rails for Zombies will teach you all the Rails basics with a zombie flair.

    • Sinatra

    A simple Ruby web framework, lighter than Ruby on Rails. Flask is inspired by this, so they're good analogues.
  • Sinatra — Official Sinatra homepage. Get started here.

    • Git

    Git is a Version Control System (VCS), which lets you track changes to code over time and collaborate with others. A VCS is critical to working on any medium-to-large application, and really helpful when working on a project with someone else.
  • Try Git — A really awesome in-browser tutorial/game that lets you get a feel for what git is like, interactively.
  • Pro Git — Pro Git, available freely on the git website, is an excellent book covering all aspects of git, including use and inner workings. A must-read to fully understand git.
  • GitHub — GitHub is a website that hosts git repositories. It lets you store a shared copy of code that you're working on with someone else, as well as see others' code and contribute to open source projects.

    • Storage (databases)

    If you want to save information when a user leaves your app or coordinate between different users, you need a database of some sort. Here are some good options to get started.
  • SQLite — A "relational" database-think about it like an Excel spreadsheet, but stricter, bigger, and more powerful. It might already be installed and it's really easy to set up a database. Note that you'll need to learn the basics of Structured Query Language (SQL) in order to use it.
  • MongoDB — A "non-relational" database. You put whole "objects" in and can query them in simple or complicated ways. Check out the interactive dmeo!
  • MariaDB (MySQL) — Another "relational" database. This one is very popular, and also powerful. You need to know SQL for this. MariaDB is based on MySQL, and was forked after the community worried that Oracle wasn't devoting enough time to the project and was hiding things; they're mostly identical, but MariaDB is the way to go for "MySQL" nowadays.

    • Hosting

    In order for your project to be out on the internet, you need to get it hosted. Here are a bunch of options to do that.
  • Neocities — This site was created to let beginners host websites with static content. Start here if you're just trying to get a web page online!
  • GitHub Pages — If you're using GitHub, this is a really easy way to publish a project page.
  • Parse — Parse is a cloud app platform with free tiers. Great place to host apps.
  • Heroku — Heroku is a Platform as a Service (PaaS) provider, which means that they host your apps for you without you having to manage any computers, virtual or otherwise.
  • Amazon Web Services — Amazon Web Services (AWS) is a cloud hosting provider that has a bunch of cloud options, the easiest of which is Elastic Cloud Compute (EC2). EC2 just gives you a virtual machine in one of their data centers that you control completely, so it's a very flexible option.

    • Java

    Java is an object oriented language, and the most widely used language in the world. It is compiled down to Java byte code, and then run on a Java Virtual Machine. Because JVMs have been made for many different platforms, people often say of Java, "Write once, run anywhere.". Java is great language if you want to program for Android or program scalable systems (both big and small companies use Java, but it's not a usual choice for prototyping). We'll probably have a hard time teaching you Java online, so these resources are more advanced.
  • Effective Java — Assumes you already know Java, and teaches you how to write really good Java.
  • Java Concurrency in Practice — Java has a battlehardened concurrency library that is very basic and widely imitated. This is the standard book on Java concurrency.

    • Android SDK

    The Android SDK is the library that lets you write programs that run on Android.
  • Android Developer Reference — Portal for all things Android development: references, tools, news.

    • Scala

    A powerful and flexible programming language, useful for tackling hard problems.
  • Try Scala — An interactive Scala demo. Try it out and try not to get hooked.
  • Scala School — How Twitter ramps up their engineers (with Java experience) on Scala.
  • Programming in Scala — Written by one of the authors of Scala, this is the definitive guide to the language.
  • Effective Scala — Twitter Scala style guide.

    • Objective-C

    A superset of C, Objective-C is the language that iOS and Mac apps are written in.
  • codeschool Try Objective-C — Learn the language that powers iOS and Mac apps. In this course, you'll learn the fundamental building blocks of Objective-C and explore the foundation of elegant and beautiful apps.
  • Big Nerd Ranch Objective-C Programming Guide — A great book that walks beginners through the basics of Objective-C. A great buy.
  • NSHipster — NSHipster is a blog that has great explanations for some of the quirks and "overlooked bits" of Objective-C and Cocoa.

    • iOS

    Learn to build an iOS application by learning to code in Objective-C.
  • Big Nerd Ranch iOS Programming Guide — A great book that walks beginners through the basics of iOS. Best after the Big Nerd Ranch Objective-C book.
  • NSHipster — NSHipster is a blog that has great explanations for some of the quirks and "overlooked bits" of Objective-C and Cocoa.
